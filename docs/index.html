To use the tool, extract the downloaded archive and run the tool. Then, follow the instructions in the tool.

Note for Windows users (click to expand)

Unfortunately, Windows Defender may flag the download as a potential threat. This is a false positive caused by some of the system calls this installer makes to set up Driver Station SD cards. If your download fails because of a "Virus Detected" message, you can follow these steps to work around this error.

  1. Open the Windows Security Settings.
  2. Click on "Virus & threat protection".
  3. Click the "Protection history" link under "Current Threats".
  4. Find the entry for the failed download. Click to expand it and double check the affected item is the setup wizard zip file.
  5. In the actions menu, select "Allow".
  6. Try downloading the file again.

When you run the application, you may be presented with a warning dialog about the safety of the application. You can bypass this dialog by clicking the "More info" link and then the "Run anyway" button.
