Gizmo Setup Wizard for BEST Robotics

This tool will help you install the software on your Gizmo system to prepare it for the BEST Robotics competition.

To use the tool, extract the downloaded archive and run the tool. Then, follow the instructions in the tool.

If you have trouble using the tool, please post a question to GitHub or reach out to Matt Barulic.
